Se desarrolló la virtualización de una planta de mezcla de pintura, para realizar un proceso de dosificado, mezcla y sellado automático. Este proyecto se realizó mediante el uso de TIA Portal para la programación de un PLC S7 1200 Siemens, Unity para la virtualización de la planta de pintura y un módulo PLC didáctico para poner a prueba la planta automatizada.
